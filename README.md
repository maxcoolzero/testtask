# Install

Clone this repository

`` `
git clone https://github.com/maxim-emelyanov/HiSmith.git
`` `

Go to project`s folder

`` `
cd HiSmith
`` `

Install required packages

`` `
npm install
`` `

Build the project

`` `
npm run build
`` `

The result will be in 'dist' folder
